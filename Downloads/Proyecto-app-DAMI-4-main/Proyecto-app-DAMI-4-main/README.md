# ClinicaCitas (Base visual)

Proyecto Android (Kotlin + Views + Gradle Kotlin DSL, minSdk 24) para el tema:
**Aplicativo móvil para control de citas en una clínica**.

## Alcance de este ZIP
- Base visual (layouts XML) y navegación entre pantallas.
- Botones funcionales.
- Login con roles usando **datos estáticos** (sin SQLite en esta fase).
- Listas (RecyclerView) con datos estáticos para **Médicos** y **Citas**.

## Roles y usuarios demo
- **ADMIN**: admin / admin123
- **PACIENTE**: paciente / paciente123

## Notas
- Los datos de médicos y citas son mock (en memoria) para fines visuales y de navegación.
